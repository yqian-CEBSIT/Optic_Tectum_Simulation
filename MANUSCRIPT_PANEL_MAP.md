# Manuscript Panel to Code Map

This map aligns the public code bundle with the bioRxiv manuscript figure package.
The scripts and notebooks in this repository are organized around the figure-level workflows and their supporting assets.

## Figure 1

- Figure 1H-J: `Figure1/Figure1_TwoPathway.ipynb`
- Input matrix: `Figure1/local_connection_prob.csv`
- Morphology/classification panels in Figure 1A-G follow the atlas-derived morphology and downstream-target annotation workflow used for the manuscript.

## Figure 2

- Figure 2D-E: `Figure2/Figure2_Simulation_WholeOT_L.ipynb` and `Figure2/Figure2_Simulation_WholeOT_S.ipynb`
- Figure 2F/I: `Figure2/accumulation/accumulation.py` with `Looming_Order.csv` or `SD_Order.csv`
- Figure 2G-H/J-K: the manuscript-aligned ablation matrices generated by `tools/generate_manuscript_assets.py`
- Canonical 52-type model files: `Figure2/neuron_number.csv` and `Figure2/neuron_connections_whole.csv`

The `noncri` matrices now mean "all remaining TIN morphotypes", matching the Figure 2 legend:

- `neuron_connections_whole_cri_L_test.csv`: ablates `I_S12` and `I_SIN`
- `neuron_connections_whole_noncri_L.csv`: ablates all TIN morphotypes except `I_S12` and `I_SIN`
- `neuron_connections_whole_cri_S_test.csv`: ablates `I_S12_S56` and `I_S12_S56_SGC`
- `neuron_connections_whole_noncri_S.csv`: ablates all TIN morphotypes except `I_S12_S56` and `I_S12_S56_SGC`

## Figure 3 and Figure S5

- All-TIN noisy-input SNR replay: `Figure3/F3_SNR_TIN_all.ipynb`
- Scripted public replay for Figure 3C-N and Figure S5 panel families: `Figure3/figure3_s5_replay.py`
- Input workbooks: `Figure3/NLooming*.xlsx` and `Figure3/NSD*.xlsx`

The public notebook supports the all-TIN noisy-input SNR workflow. `figure3_s5_replay.py` adds public entry points for:

- Figure 3C/J style response curves
- Figure 3D/K e-TIN-only and i-TIN-only comparisons relative to no-TIN controls
- Figure S5 morphotype-by-morphotype fold-change scans at selected noise levels
- Figure 3E/L robustness-score ranking and top-four e-TIN selection
- Figure 3F/G/M/N style top-four versus non-specific e-TIN ablation summaries

## Figure 4 and Figure S6

- Main 10-degree BMD notebook: `Figure4/F4_BD_Total.ipynb`
- Component-blocking notebook: `Figure4/F4_BD_remove.ipynb`
- Manuscript-formula replay script: `Figure4/figure4_bmd_replay.py`
- Machine-readable Table S3 input: `Figure4/serotonergic_connections.csv`
- 10-degree BMD input: `Figure4/BD_10.xlsx`, sheet `Sheet1`
- 16-degree BMD input for Figure S6: `Figure4/BD_16.xlsx`, sheet `smoothed_repaired`

The replay script implements the STAR Methods threshold formula:

```text
Vth_i(5HT) = Vth_i - gamma * p_i
```

where `p_i` is read from `serotonergic_connections.csv`.

Example smoke command with a shortened simulation:

```bash
python Figure4/figure4_bmd_replay.py --subtype dl --sim-duration-ms 1000 --analysis-start-ms 200 --analysis-end-ms 800
```

Full Figure 4/S6 runs should use the default 60 s simulation and 30-45 s analysis window.
